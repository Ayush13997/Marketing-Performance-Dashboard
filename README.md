# Marketing-Performance-Dashboard

Overview
An interactive Power BI dashboard designed to track and analyze marketing campaign performance across five states. The dashboard processes over 200,000 rows of campaign data and provides real-time insights into key marketing metrics.

Tools Used
Power BI

Power Query

Excel

Dataset Columns
Column	Description
Campaign_ID	Unique identifier for each campaign
Company	Company name
Campaign_Type	Type of marketing campaign
Target_Audience	Audience segment targeted
Duration	Campaign duration in days
Channel_Used	Marketing channel (Google Ads, Email, Website, Instagram, YouTube, Facebook)
Conversion_Rate	Percentage of conversions
Acquisition_Cost	Cost to acquire customer
ROI	Return on investment
Location	City (Chicago, Houston, Los Angeles, Miami, New York)
Language	Language used
Clicks	Number of clicks
Impressions	Number of impressions
Engagement_Score	Engagement metric
Customer_Segment	Customer segment category
Date	Campaign date
Key Features
Interactive Filters
Clickable year filter buttons to view performance data by specific years

Geographic filters to analyze campaigns across five cities

Dynamic dashboard actions that update all visualizations based on user selection

Calculated Metrics
Click Through Rate – Measures campaign engagement

Revenue – Derived from Acquisition Cost × Return on Investment

Cost Per Click – Calculates efficiency of ad spend

Visualizations
Revenue and Ad Spend trends over time

Channel-wise impressions comparison

Top 10 performing campaigns with detailed metrics

Geographic performance analysis

Month-over-month variation tracking
